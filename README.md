# nba_mvp_data_crawl

The main predict code is logreg2.py, works in both Python2 and Python3
## Dependencies
- pandas
- sklearn

## Parameters
- --year from 2000 to 2018
- --model model1 for the top-down model, model2 for the bottom-up model

## Example
```
$ python logreg2.py --year 2018 --model model1
```
